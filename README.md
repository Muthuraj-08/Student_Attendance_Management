# Student_Attendance_Management
A simple and efficient console-based Attendance Management System developed in C programming language. This project allows colleges or schools to maintain and track the attendance of students using a menu-driven interface.
  
This project allows colleges or small institutions to:

- Add students  
- Mark daily attendance  
- View attendance reports with percentages  

This project is beginner-friendly and ideal for **C programming practice and mini-project submissions**.
